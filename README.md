# ytc
an YouTube Client built with tauri and yew (both in rest)

# build instructions

### install tauri
```console
npm install
```

### dev app with hot reload
```console
npm run tauri dev
```

### release app
```console
npm run tauri build
```
the build is available at `src-tauri/target/release`
